# Evaluador de Expresiones

Aplicación web con frontend y backend.

## Requisitos

- Python 3.13.12
- Node.js 24.13.0

## Backend

cd backend
pip install -r requirements.txt
uvicorn main:app --reload

## Frontend

cd frontend
npm install
npm run dev

## Uso

1. Abrir la web
2. Ingresar JSON y expresión
3. Validar o evaluar
4. Ver resultado
